How to Play

Controls

* drag the screen with the left mouse to rotate
* zoom in and out using the scroll wheel

Commands

fill command

fill:(x1, y1, z1, x2, y2, z2, block)

save command

save (for now world save code will log to console)

Load command

load:`enter worlds save code here`

world save code is generated when you save the world and does not have qoutation marks around when enetering into world load command

Place tree command

tree:(x,y,z,type)

* only oak trees available now
* generated trees are randomly given differnt shape
